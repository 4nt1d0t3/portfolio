The Anti-Bug Project!

My first ever independent project built using node.js, express, mySQL and styled with Bootstrap4 and some custom CSS

The Anti-Bug is a simple project and bug tracking web app that uses RESTful routing and a SQL database to keep track of users, projects and bugs. During this project I learnt a lot more about how to build projects in general using MVC architecture, working with SQL datbases and that I am terrible with puns.
